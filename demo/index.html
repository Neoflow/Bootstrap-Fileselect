Bootstrap Fileselect

Examples

Basic

The plugin will replace the selected input field with a bootstrap-styled file-select inputfield to get the same designs like for the other form controls.

<input id="demo1" type="file" name="demo" />
<script>
    $('#demo1').fileselect();
</script>

Multiple

Same behaviour as mentioned in the basic example, but you can choose multiple files now.

<input id="demo2" type="file" name="demo" multiple />
<script>
    $('#demo2').fileselect();
</script>

Validation

You can set two types of validation rules:

  • Limit the number of selectable files
  • Restrict the allowed file extensions
<input id="demo3" type="file" name="demo" multiple />
<script>
    $('#demo3').fileselect({
        limit: 3, // default: false, no limitation
        extensions: ['jpeg', 'jpg', 'png'] // default: false, all extensions allowed
    });
</script>

Styling

You can define the css class and icon of the browse button.

<input id="demo4" type="file" name="demo" multiple />
<script>
    $('#demo4').fileselect({
        browseBtnClass: 'btn btn-danger', // default: btn btn-primary
        browseIcon: '<i class="glyphicon glyphicon-hdd"></i>' // default: <i class="glyphicon glyphicon-folder-open"></i>
    });
</script>

Translations

The plugin will use the browser language, but you can also define a language. If the defined language isn't supported, then the plugin will fallback to the english translations.

<input id="demo5" type="file" name="demo" multiple />
<script>
    $('#demo5').fileselect({
        language: 'de' // default: false, browser language
    });
</script>
